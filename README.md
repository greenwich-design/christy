# Christy (GD) 2023

A theme build for [https://www.christy.co.uk](https://www.christy.co.uk/)

# Setup

1) Install Laravel Mix:
<br><br>
npm init -y<br>
npm install laravel-mix --save-dev
<br><br>
2) Install TailwindCSS
<br><br>
npm install -D tailwindcss<br>
npx tailwindcss init

3) Build/Watch<br><br>

development:<br>
npx mix watch
<br><br>
build:<br>
npx mix --production
